# Chart Analyzer

TradingView-style charting application with event-based saving and technical indicators.

## Features

- Interactive candlestick charts with Lightweight Charts
- Multiple timeframes and chart types
- Technical indicators (SMA, EMA, RSI, MACD, Bollinger Bands)
- Event-based chart saving with full state preservation
- Chart library with filtering and timeline view
- Dark professional UI inspired by TradingView

## Tech Stack

- Next.js 14+
- TypeScript
- Tailwind CSS
- Lightweight Charts
- React 18+
